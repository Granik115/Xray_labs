# X-Ray-lab

Лабораторные работы по рентгенотехнике (рентгеноабсорбционный метод оценки концентрации).

Приложение для Windows 10/11. Python + PyQt5. Интерфейс в цветовой гамме MolPlayer (см. Laby.docx).

## Возможности (v0.0.7)

- Главное окно 800×600: выбор лабораторной работы (10 строк, реализована Л/р 1).
- Л/р 1 (1000×660): оценка концентрации вещества рентгеноабсорбционным методом.
  - Открытие JPEG/BMP снимков.
  - Тип контейнера (квадратный / цилиндрический).
  - Интерактивная линия калибровки (ЛКМ + перетаскивание).
  - Ввод размера (1–99 мм) и толщины слоя.
  - Тип включений (кубические / шарообразные) — влияет на формулы.
  - «Найти включения» — локальный анализ контраста с наложением результата на снимок.
  - «Рассчитать объёмы» с проверкой данных.
  - Кнопки «Методика» и «Протокол» (PDF из `resources/`).
- Встроенный менеджер версий: просмотр всех GitHub Releases, обновление,
  переустановка и откат на выбранную версию.
- Обновление установленной версии проходит полностью автоматически и без окон
  мастера установки.
- Проверка SHA-256, безопасная распаковка и восстановление portable-версии при
  ошибке копирования.
- Установщик Windows (Inno Setup): выбор папки, ярлык, удаление через «Программы».

## Установка в Windows 10/11

Рекомендуемый вариант — скачать
[`Xray_labs-0.0.7-setup.exe`](https://github.com/Granik115/Xray_labs/releases/download/v0.0.7/Xray_labs-0.0.7-setup.exe),
запустить его и пройти обычный мастер установки. Права администратора не нужны:
программа устанавливается в профиль текущего пользователя.

Если Windows SmartScreen покажет предупреждение для неподписанного приложения,
нажмите «Подробнее» → «Выполнить в любом случае». Релизные файлы дополнительно
проверяются контрольными суммами из `SHA256SUMS.txt`.

После установки кнопка `↻` в заголовке приложения открывает полный список версий.
Можно выбрать более новую, текущую или старую версию; приложение само
подберёт установщик либо portable-пакет.

## Запуск из исходников

```powershell
cd C:\projects\Xray_labs
python -m pip install -r requirements.txt
python main.py
```

## Сборка

```powershell
cd C:\projects\Xray_labs
.\build.bat
```

Результаты в папке `releases/` (не коммитится в git):

| Файл | Назначение |
|------|------------|
| `Xray_labs-0.0.7-setup.exe` | **Установщик** — прикрепляйте к GitHub Release |
| `Xray_labs-v0.0.7-portable.zip` | Портативная версия (с номером версии) |
| `Xray_labs-portable.zip` | Копия для авто-обновления в приложении |

Для установщика нужен [Inno Setup 6](https://jrsoftware.org/isinfo.php). Без него соберётся только portable zip.

При публикации изменения в `main` GitHub Actions автоматически проверяет код,
собирает Windows-пакеты и создаёт релиз для версии из `constants.py`, если такого
тега ещё нет.

## Редактирование интерфейса в Qt Designer

- `ui/main_window.ui` — главное окно
- `ui/lab1_window.ui` — окно Л/р 1

Откройте `.ui` в Qt Designer, сохраните — код подхватит изменения через `uic.loadUi`.
