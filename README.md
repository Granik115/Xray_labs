# X-Ray-lab

Лабораторные работы по рентгенотехнике (рентгеноабсорбционный метод оценки концентрации).

Приложение для Windows 10/11. Python + PyQt5. Интерфейс в цветовой гамме MolPlayer (см. Laby.docx).

## Возможности (v0.0.2)

- Главное окно 800×600: выбор лабораторной работы (10 строк, реализована Л/р 1).
- Л/р 1 (1000×600): оценка концентрации вещества рентгеноабсорбционным методом.
  - Открытие JPEG/BMP снимков.
  - Тип контейнера (квадратный / цилиндрический).
  - Интерактивная линия калибровки (ЛКМ + перетаскивание).
  - Ввод размера (1–99 мм) и толщины слоя.
  - Тип включений (кубические / шарообразные) — влияет на формулы.
  - «Найти включения» — обработка (оригинал + результат рядом).
  - «Рассчитать объёмы» с проверкой данных.
  - Кнопки «Методика» и «Протокол» (PDF из `resources/`).
- Обновление и откат версий с GitHub Releases (portable zip).
- Установщик Windows (Inno Setup): выбор папки, ярлык, удаление через «Программы».

## Запуск из исходников

```powershell
cd C:\projects\Xray_labs
python -m pip install -r requirements.txt
python main.py
```

## Сборка

```powershell
cd C:\projects\Xray_labs
.\build.bat
```

Результаты в папке `releases/` (не коммитится в git):

| Файл | Назначение |
|------|------------|
| `Xray_labs-0.0.2-setup.exe` | **Установщик** — прикрепляйте к GitHub Release |
| `Xray_labs-v0.0.2-portable.zip` | Портативная версия (с номером версии) |
| `Xray_labs-portable.zip` | Копия для авто-обновления в приложении |

Для установщика нужен [Inno Setup 6](https://jrsoftware.org/isinfo.php). Без него соберётся только portable zip.

## Редактирование интерфейса в Qt Designer

- `ui/main_window.ui` — главное окно
- `ui/lab1_window.ui` — окно Л/р 1

Откройте `.ui` в Qt Designer, сохраните — код подхватит изменения через `uic.loadUi`.
