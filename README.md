# X-Ray-lab

Лабораторные работы по рентгенотехнике (рентгеноабсорбционный метод оценки концентрации).

Приложение для Windows. Python + PyQt5. Интерфейс в точной цветовой гамме MolPlayer (промышленный молекулярный преобразователь).

## Возможности (v1.0.0)

- Главное окно: выбор лабораторной работы (пока реализована Л/р 1).
- Л/р 1: Оценка концентрации вещества рентгеноабсорбционным методом.
  - Открытие jpeg/bmp снимков.
  - Выбор типа контейнера (квадратный / цилиндрический) — меняет подсказки и формулы.
  - Интерактивное выделение линии диаметра/стороны на снимке (ЛКМ + перетаскивание).
  - Ввод реального размера (мм) и толщины слоя.
  - Выбор типа включений.
  - «Найти включения» — обработка изображения (оригинал + обработанное рядом): вне контура серый, контур чёрный, включения белые.
  - «Рассчитать объемы» с проверкой данных + сообщение «Котик, не ходи мимо лотка...».
  - Вычисление объёмов породы (см³) и включений (мм³) по описанной в ТЗ логике (формулы частиц — заглушка, ждёт точных из методики).
  - Кнопки «Методика» и «Протокол» (плейсхолдеры + сообщение; положите PDF в resources/).
- Полупрозрачность окна (~0.93), тёмная сине-голубая тема 1:1 из MolPlayer.
- Шрифт Comic Sans MS (с fallback).
- Маленькая кнопка проверки обновлений в углу.
- Авто-проверка обновлений при запуске (тихая).
- Самообновление из GitHub Releases (portable zip).
- Сборка в портативный exe (как в MolPlayer).

## Запуск из исходников

```powershell
cd C:\projects\Xray_labs
python -m pip install -r requirements.txt
python main.py
```

## Сборка портативной версии и exe (рекомендуется onedir)

```powershell
cd C:\projects\Xray_labs
.\build.bat
# или
powershell -ExecutionPolicy Bypass -File build.ps1
```

После сборки:
- Портативная папка: `dist\Xray_labs\`
- Архивы для релиза: `releases\Xray_labs-v1.0.0-portable.zip` и `Xray_labs-portable.zip` (для авто-обновлений)

**Как раздать / первый релиз:**
1. Запусти build.ps1
2. Возьми файлы из `releases/`
3. На GitHub Releases создай релиз v1.0.0
4. Прикрепи оба zip (versioned + stable portable).
5. Обновления в приложении ищут ассет с "portable" в имени + .zip

## Место под PDF (Методика и Протокол)

Положите файлы:
- `resources/metodika.pdf`
- `resources/protokol.pdf`

Кнопки в приложении откроют их (QDesktopServices) или покажут понятное сообщение, если файлов нет. Можно вшить позже через pyinstaller datas или Qt resources.

## Структура проекта (важно для .ui)

- `ui/main_window.ui` и `ui/lab1_window.ui` — редактируйте в **Qt Designer**! 
- Код загружает их через `PyQt5.uic.loadUi` (никакого ручного редактирования сгенерённого .py не нужно для визуала).
- Кастомная логика (линия, обработка изображения) — в Python (QGraphicsView + eventFilter + Pillow).

## Цвета

Взяты полностью из MolPlayer/constants.py (см. там комментарии про референс molecular transformer).
